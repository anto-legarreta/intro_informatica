## Instalación de VSCode

VSCode es un IDE (entorno de desarrollo integrado) de Microsoft. 
Como IDE nos facilita la creacion de programas o aplicaciones.
En el Campus Virtual de la carrera se encuentra una guía de instalación.

## La terminal

La terminal es una interfaz por la que podemos darle órdenes a la computadora en forma de lineas de texto.
Nos permite hacer todo lo que podríamos hacer con una interfaz y más. Crear una carpeta, borrar un archivo, renombrarlo, etc.

## Comandos

Algunos comandos básicos son:

`ls` Muestra los archivos de la carpeta en la que estamos ubicados

`cd ..` Nos permite retroceder una carpeta

`rm nombreArchivo.extensión` Elimina el archivo que le indicamos

`mv nombreAnterior nombreNuevo` Vambia el nombre de un archivo

`clear` Limpia todo lo que hayamos escrito en la terminal

`mkdir nombreCarpeta` Crea una carpeta con el nombre que le indiquemos

Existen muchos comandos para la terminal, que nos permiten realizar diversas tareas.

## Actividad de la clase sincrónica

```
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio$ mkdir Raiz
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio$ cd Raiz/
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz$ mkdir carpeta1 carpeta2 carpeta3
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz$ cd carpeta1

anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz/carpeta1$ cd ..
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz$ cd carpeta3
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz/carpeta3$ touch Introduccion.txt Informatica.txt Eliminar.txt
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz/carpeta3$ cd ..
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz$ rm carpeta1/Eliminar.txt carpeta3/Eliminar.txt 
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz$ ls carpeta1
Digital.txt  House.txt
anto-legarreta@antolegarreta-VirtualBox:~/Escritorio/Raiz$ ls carpeta3

```
