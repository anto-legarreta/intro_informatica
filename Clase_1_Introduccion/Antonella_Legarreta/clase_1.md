## ¿Qué es una computadora?

La computadora es un dispositivo capaz de recibir ordenes y ejecutarlas. Este dispositivo ha evolucionado a lo largo de la historia, llevandola a lo que hoy conocemos.

Con IBM se entra en la "era cognitiva" donde encontramos computadoras cuánticas, que pueden resolver calculos complejos que a una computadora "común" le llevaria aproximadamente diez mil años.

## Glosario técnico

* Hardware
  * Entrada: Componentes que permiten el ingreso de información
  * Salida: Componentes que permiten la salida de la información
  * Internos: Componentes que forman parte del dispositivo y son escenciales para su funcionamiento

* Software
  * Sistemas: Desvincula al usuario y al programador de los detalles del sistema informatico que se use, como el procesamiento de memoria, discos, etc... Provee al usuario y al programador de interfaces de alto nivel.
  * Programación: Herraminetas qu epermiten al programador desarrollar programas informáticos.
  * Aplicación: Permite a los usuarios llevar a cabo una o varias tareas específicas, donde quiera que el proceso pueda ser automatizado o asistido, sobretodo para negocio. 

* Servidores
  * Web: Almacena documentos y distribuye el contenido a 'clientes' que lo piden en la red.
  * Base de Datos: Provee servicios de base de datos a programas o computadoras. Tambien pueden estar dedicadas a prestar servicio ejecutando programas.
  * Archivos: Almacena archivos y los distribuye en la red.

* Interfaz de usuario
  * Interfaz de linea de comandos: Interprete de comando sólo de texto.
  * Intergaz grafica: Permite dar ordenes a la maquina mediante un medio grafico intuitivo.
  * Interfaz nativa: Permite comunicarse con el dispositivo sin utilizar sistemas de mando o dispositivos de entrada

Hardware: | El hardware o equipo se refiere a las partes físicas, tangibles, de un sistema informático, sus componentes eléctricos, electrónicos, electromecánicos y mecánicos. Los cables, así como los gabinetes, los periféricos de todo tipo, y cualquier otro elemento físico involucrado; contrariamente, el soporte lógico e intangible es el llamado software.
Software:|Se conoce como software o soporte lógico al sistema formal de un sistema informático, que comprende el conjunto de los componentes lógicos necesarios que hacen posible la realización de tareas específicas, en contraposición a los componentes físicos que son llamados hardware. La interacción entre el software y el hardware hace operativo un ordenador (u otro dispositivo), es decir, el software envía instrucciones que el hardware ejecuta, haciendo posible su funcionamiento.
Servidores:|Un servidor es una computadora capaz de atender las peticiones de un cliente y devolverle una respuesta en concordancia. Se pueden ejecutar en computadoras dedicadas a las cuales se les conoce individualmente como “el servidor” o en la mayoría de los casos una misma computadora puede proveer múltiples servicios y tener varios servidores en funcionamiento.
interfaces nativas:|son las que vienen ya hechas para facilitarte un poco el trabajo a la hora de crear la aplicación. Tienen un aspecto ya definido en cuanto a color, medida o tipografía.
Interfaz gráfica de usuario:|La interfaz gráfica de usuario, conocida también como GUI (del inglés graphical user interface), es un programa informático que actúa de interfaz de usuario, utilizando un conjunto de imágenes y objetos gráficos para representar la información y acciones disponibles en la interfaz. Su principal uso consiste en proporcionar un entorno visual sencillo para permitir la comunicación con el sistema operativo de una máquina o computador. Usado para ingresar comandos. En los primeros días de la informática, antes del ratón, era la forma estándar de interactuar con una computadora. Mientras que la interfaz gráfica de usuario (GUI) ha reemplazado en gran medida a las CLI, todavía se incluyen con varios sistemas operativos, como Windows y OS X.
