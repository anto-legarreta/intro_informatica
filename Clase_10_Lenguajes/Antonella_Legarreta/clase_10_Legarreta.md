# Lenguajes de programación

Estos lenguajes nos permiten transmitirle a la computadora órdenes no ambiguas para realizar tareas.

## Tipado

* Débil: En este tipo de lenguajes no se suele indicar el tipo de variable. Podemos asignar un valor entero a una variable que anteriormente tenia una cadena, pero, además, podemos operar con variables de distintos tipos.
* Fuerte: Nos obliga a indicar el tipo de dato al declarar la variable, y dicho dato no puede ser cambiado una vez definida la variable.
* Estático: La comprobación de la tipificacion se realiza durante la compilacion.
* Dinámico: La comprobación de tipificación se realiza durante su ejecución.

## Frameworks ~ Marco de trabajo

Estructura previa que se puede aprovechar para realizar un proyecto. Simplifica las tareas ya que se aplica complementando lo que se quiere realizar.

## Paradigmas de programación

* Imperativa: Se le indica a la computadora que debe hacer, en que orden y a traves de instrucciones sucesivas.
  * Estructurada
  * Orientada a objetos
* Multiparadigma: Lenguajes que pueden usar mas de un paradigma
* Declarativa: Describe el resultado que se busca
  * Logica
  * Funcional
* Especifico de domino: Lenguajes desarrollados para resolver un problema espefico

## Paradigmas

* Estructurado: Se ejecuta una instruccion a la vez y se disponen de pocas instrucciones.
* Orientado a objetos: El código se puede agrupar de tal forma que se represente una entidad y que interprete mensajes.
* Funcional: Se basa en funciones matemáticas.
* Lógico: En vez de desarrollar instrucciones se utulizan reglas logicas para consultar al sistema y éste infiere que hacer en base a la lógica establecida
* Especifico de dominio: Resuelves problematicas muy específicas.
* Multiparadigma: Permiten el uso de varios paradigmas mencionados anteriormente.
