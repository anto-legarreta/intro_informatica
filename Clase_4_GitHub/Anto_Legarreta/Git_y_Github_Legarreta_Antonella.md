## Práctica individual obligatoria

1. ¿Quién inventó el sistema de control de versión Git y por qué?

    Fue inventado por Linus Torvalds (creador de kernel linux), con el proposito de crear un sistema de registro de versiones para aplicaciones con mucho codigo fuente.

2. ¿A quién pertenece actualmente Github y por qué?
  
    GitHub pertenece actualmente a la empresa Microsoft.
    
    La CEO de Microsoft dijo:
    
    Microsoft es una empresa que prioriza a los desarrolladores, y al unir fuerzas con GitHub se fortalece nuestro conmpromiso para con su libertad, e innovación.
    La compañía aseguró que GitHub funcionará como una unidad independiente, y que el plan es empoderar a los desarrolladores a la vez brindando herramientas para nuevas audiencias.
    
    [Fuente](https://www.zdnet.com/article/microsofts-github-acquisition-its-all-about-developer-relationships-influence/#:~:text=Microsoft%20is%20a%20developer%2Dfirst,its%20tools%20to%20new%20audiences.)
