# Estructura y tecnología de las computadoras

## Componentes de una computadora

### Internos

Estos elementos son inseparables de la computadora ya que si faltara alguno ésta no funcionaría.

* Placa madre / motherboard: A través de esta placa todos los dispositivos se conectan tanto de manera directa como indirecta.
* Procesador / CPU: Interpreta y ejecuta instrucciones a través de operaciones básicas. Dirigen las operaciones que realiza la computadora.
* Memoria RAM: Almacena informacion de forma termporal y su contenido se elimina cuando la computadora se apaga.
* Memoria ROM: Almacena la información de manera permanente, y guarda toda la configuracion inicial para que arranque la computadora con sus funciones básicas.
* Placa de video: Se encarga de mostrar las imágenes por el monitor.
* Placa de sonido: Permite a la computadora reproducir y recibir sonidos a traves de los correspondientes dispositivos de entrada y salida.
* Almacenamiento secundario: Estos almacenan la información de manera permanente, que no es necesaria para el funcionamiento de la computadora. Pueden ser archivos de todo tipo. Este dispositivo es el disco rígido o disco duro.

### Externos

Estos dispositivos pueden ser usados pero no son escenciales para el funcionamiento de la computadora

* De entrada: Introducen datos a la computadora
* De salida: Extraen dato de la computadora.
* Mixtos: Cumplen ambas funciones (entrada y salida)
* Almacenamiento: Permiten almacenamiento permanente. Son de conexión externa.
* Comunicación: Permiten conexión entre computadoras.

## La CPU

El microprocesador es el cerebro del sistema ya que procesa todo lo que ocurre en la computadora y ejecuta las acciones.

Se comunica con la RAM para leer datos e instrucciones. Se comunica con los dispositivos de entrada y de salida para recibir datos y mostrarlos.

Las CPU tienen nucleos, existen las de uno solo, que no pueden realizar mas de una tarea al mismo tiempo, y las de multiples nucleos que permiten realizar varias, según la cantidad de núcleos.

Los hilos son el flujo de control de programa. Ayudan a que el procesador administre sus tareas aprovechando mejor los "tiempos de espera".

La caché del CPU es una memoria dentro de la CPU y almacena datos e instrucciones muy utilizadas para que los procesos sean mas rápidos. Esta memoria tiene poca capacidad pero es muy rápida, de esta forma los procesos se realizan mas rapido al no tener que ir a buscar la información hasta la RAM para poder realizarlo.

## Ejercicio sincrónico

Requisitos Office:
1.6 GHz o más rápido, dos núcleos

Requisitos Filmora:
Intel i3 o superior multinúcleo, 2 GHz o superior

Requisitos Cyberpunk 2077:
Intel Core i5-3570K o AMD FX-8310

Con estos datos, el procesador más conveniente que puede cumplir con los tres requisitos es el Intel Core i7-4790. El Cyberpunk 2077 no puede correr en ese modelo de i5.
